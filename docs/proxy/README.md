## 📖 核心命令速查

```bash
# 启用代理
proxy

# 检查代理状态
check_proxy --status

# 显示完整配置
proxy_status

# 禁用代理
unproxy

# 自定义代理地址
proxy 127.0.0.1:10808 --verify
```
